FarmaPocket

Política de Privacidade

Última atualização: 01 de julho de 2026

1. Introdução

O FarmaPocket é um aplicativo de gestão pessoal de medicamentos. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

Importante: O FarmaPocket não é um serviço de saúde. Não fornecemos diagnósticos, prescrições ou orientações médicas. O app é uma ferramenta de organização pessoal.

2. Dados que Coletamos

Coletamos apenas os dados necessários para o funcionamento do app:

  • Dados de conta: nome, e-mail (via Google OAuth), foto de perfil
  • Dados de dependentes: nome, data de nascimento, relacionamento
  • Dados de saúde: nomes de medicamentos, princípios ativos, dosagens, frequências, sintomas, tratamentos, eventos médicos, receituários
  • Dados operacionais: quantidade em estoque, datas de início/término, profissionais de saúde
  • Dados técnicos: data/hora de acesso, tipo de dispositivo (para melhorias)

3. Como Usamos Seus Dados

Seus dados são usados exclusivamente para:

  • Exibir e organizar seus medicamentos e tratamentos
  • Calcular autonomia de estoque e alertar sobre vencimentos
  • Enviar lembretes de medicamentos (se ativado)
  • Permitir exportação de backup dos seus dados
  • Melhorar a estabilidade e usabilidade do app

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins comerciais.

4. Armazenamento e Segurança

Seus dados são armazenados na Supabase, um serviço de banco de dados PostgreSQL. Utilizamos:

  • Criptografia TLS/SSL para transmissão de dados (HTTPS)
  • Row Level Security (RLS) - cada usuário acessa apenas seus próprios dados
  • Autenticação via Google OAuth - sem senhas próprias
  • Backups automáticos pelo Supabase

Transferência Internacional: Os servidores da Supabase podem estar localizados fora do Brasil (EUA ou outras regiões). A Supabase garante proteção dos dados mediante Cláusulas Contratuais Padrão (SCCs) e adesão a frameworks de proteção de dados internacionais.

5. Dados de Terceiros (Dependentes)

Se você cadastrar dependentes (filhos, cônjuge, etc.), você é responsável por obter o consentimento dessas pessoas (ou de seus responsáveis legais, no caso de menores) para armazenar seus dados no app. Recomendamos que informe os dependentes sobre o uso do FarmaPocket.

6. Seus Direitos (LGPD)

Você tem os seguintes direitos:

  • Acesso: solicitar cópia dos seus dados
  • Retificação: corrigir dados incorretos
  • Exclusão: apagar permanentemente sua conta e todos os dados
  • Portabilidade: exportar seus dados em formato JSON
  • Revogação: cancelar o uso do app a qualquer momento

Para exercer seus direitos, acesse as Configurações do app ou entre em contato pelo e-mail abaixo.

7. Exclusão de Conta

Você pode excluir sua conta a qualquer momento através do menu Configurações > Excluir Conta. Esta ação é irreversível e removerá permanentemente todos os seus dados, incluindo medicamentos, tratamentos, dependentes e receituários.

8. Uso de Cookies e Tecnologias Similares

O FarmaPocket utiliza localStorage e IndexedDB para armazenar dados localmente no seu dispositivo (funcionamento offline). Não utilizamos cookies de rastreamento ou publicidade.

9. Serviços de Terceiros

Utilizamos os seguintes serviços:

  • Google OAuth: para autenticação. Sujeito às políticas do Google.
  • Supabase: para banco de dados e autenticação. Sujeito às políticas da Supabase.
  • Netlify: para hospedagem do frontend. Sujeito às políticas da Netlify.

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos você sobre mudanças significativas através do app ou e-mail. O uso continuado do FarmaPocket após alterações constitui aceitação da nova política.
